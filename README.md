# baik-cli-hello-world

## Description
A POC for baik programming language.

## Tech stack
- gcc

## Docker stack
- ubuntu:22.04
- rzuckerm/baik

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
